# 46組合

## MKU規則
從 n 個不同元素中選出 r 個，只考慮「選哪些」，不考慮順序。

## 題型
- 從多人中選出代表 / 委員 / 組員
- 選擇題：從多個物品中挑選幾個
- 分組但組內不看順序
- 與排列比較（題目不在意順序）

## 例題
題目：
從 8 個學生中選出 3 人當代表，有幾種選法？

## 解題
步驟：
1. 判斷是否在意順序（代表ABC和CBA相同 → 不在意）
2. 使用組合公式  
   C(n,r) = n! / (r!(n-r)!)
3. 計算  
   C(8,3) = 8! / (3!5!)  
   = (8×7×6)/(3×2×1)  
   = 56
