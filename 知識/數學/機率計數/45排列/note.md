# 45排列

## MKU規則
從 n 個不同元素中選出 r 個並「依順序排列」，順序不同視為不同情況。

## 題型
- 排名 / 排隊 / 座位安排
- 從多人中選出幾人並排出順序
- 數字排列組成數字
- 需要考慮順序的選擇問題

## 例題
題目：
從 7 個人中選出 3 人排成一列，有幾種排列？

## 解題
步驟：
1. 判斷是否在意順序（在意 → 排列）
2. 使用公式  
   P(n,r) = n! / (n-r)!
3. 計算  
   P(7,3) = 7! / 4!  
   = 7×6×5  
   = 210
