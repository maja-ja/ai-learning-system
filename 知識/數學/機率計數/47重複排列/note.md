# 47重複排列

## MKU規則
當 n 個位置，每個位置都可以從 k 種元素中選擇，且元素可以重複使用時，排列數為 k^n。

## 題型
- 密碼 / 編號 / 序列生成
- 每個位置可重複選擇的排列
- 字母或數字組成的字串
- 函數或映射問題（每個位置獨立選擇）

## 例題
題目：
用 0~9 的數字組成 4 位數密碼（允許重複），共有幾種？

## 解題
步驟：
1. 判斷是否允許重複（允許）
2. 判斷每個位置選擇數（10 種）
3. 判斷位置數量（4 個）
4. 使用公式  
   k^n
5. 計算  
   10^4 = 10000
