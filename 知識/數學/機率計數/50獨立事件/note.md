# 50獨立事件

## MKU規則
若事件 A 的發生不影響事件 B 的機率，則 A、B 為獨立事件，且  
P(A ∩ B) = P(A) × P(B)

## 題型
- 多次擲硬幣 / 擲骰子
- 重複隨機實驗
- 機率連續發生問題
- 判斷事件是否互相影響

## 例題
題目：
擲兩次公平硬幣，兩次都是正面的機率是多少？

## 解題
步驟：
1. 判斷兩事件是否互相影響（不影響 → 獨立事件）
2. 計算單次機率  
   P(正面) = 1/2
3. 使用公式  
   P(A ∩ B) = P(A) × P(B)
4. 計算  
   (1/2) × (1/2) = 1/4
