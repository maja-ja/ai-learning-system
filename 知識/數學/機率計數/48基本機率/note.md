# 48基本機率

## MKU規則
事件 A 的機率為「有利結果數 ÷ 全部可能結果數」。

## 題型
- 擲骰子 / 擲硬幣
- 抽球問題
- 基本隨機實驗
- 作為條件機率或期望值的基礎

## 例題
題目：
擲一顆公平骰子，出現偶數的機率是多少？

## 解題
步驟：
1. 列出全部可能結果  
   {1,2,3,4,5,6}
2. 找出符合事件的結果  
   {2,4,6}
3. 使用公式  
   P(A) = 有利結果 / 全部結果
4. 計算  
   P = 3/6 = 1/2
