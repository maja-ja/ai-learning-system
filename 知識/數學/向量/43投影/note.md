# 43投影

## MKU規則
向量 A 在向量 B 方向上的投影長度為  
|A|cosθ ，向量投影為  
proj_B(A) = (A·B / |B|²) B

## 題型
- 向量在某方向上的分量
- 兩向量夾角問題
- 力在某方向上的分解
- 點積與幾何關係題

## 例題
題目：
已知 A=(3,4)，B=(1,0)，求 A 在 B 方向上的投影長度。

## 解題
步驟：
1. 計算內積  
   A·B = 3×1 + 4×0 = 3
2. 計算 |B|  
   |B| = 1
3. 投影長度公式  
   A·B / |B|
4. 計算  
   3 / 1 = 3
